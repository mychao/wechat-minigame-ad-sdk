# 微信小游戏广告配置说明

## 一、广告位配置

### 1.1 配置文件位置

广告位 ID 应在游戏初始化时进行配置，建议在 `GameManager.ts` 或专门的配置文件中设置。

### 1.2 配置方式

```typescript
import { AdManager } from './docs/wechat-ads/AdManager';

// 在游戏初始化时配置
AdManager.getInstance().configure({
    banner: "adunit-xxxxxx",           // Banner 广告位 ID
    interstitial: "adunit-bbbbbb",     // 插屏广告位 ID
    rewardedVideo: {                     // 激励视频按位置配置
        "gold": "adunit-yyyyyy",      // 位置 "gold" 使用广告位 yyyyyy
        "revive": "adunit-zzzzzz",    // 位置 "revive" 使用广告位 zzzzzz
        "speedUp": "adunit-aaaaaa",   // 位置 "speedUp" 使用广告位 aaaaaa
        "unlock": "adunit-yyyyyy",    // 位置 "unlock" 可以复用同一个广告位
    },
});
```

### 1.3 位置与广告位关系

每个位置（场景）对应一个固定的广告位 ID：

| 位置标识 | 说明 | 广告位 ID |
|---------|------|-----------|
| `"gold"` | 看视频获得金币 | adunit-yyyyyy |
| `"revive"` | 复活功能 | adunit-zzzzzz |
| `"speedUp"` | 加速功能 | adunit-aaaaaa |
| `"unlock"` | 解锁关卡 | adunit-yyyyyy（可复用） |

> **注意**：不同位置可以配置相同的广告位 ID（如上方 gold 和 unlock 共用）。

### 1.4 动态设置某个位置的广告位

```typescript
// 动态为某个位置设置/更新广告位
AdManager.getInstance().setRewardedVideoAdUnit("gold", "adunit-new-gold");

// 获取某个位置的广告位 ID
const adUnitId = AdManager.getInstance().getRewardedVideoAdUnit("gold");
console.log(adUnitId); // "adunit-new-gold"
```

### 1.5 测试广告位 ID

开发阶段可使用微信官方提供的测试广告位 ID：

```typescript
// 测试广告位 ID（仅供开发测试，无需开通流量主）
const TEST_AD_UNITS = {
    banner: "adunit-0000000000000000",
    interstitial: "adunit-0000000000000003",
    rewardedVideo: {
        "gold": "adunit-0000000000000001",
        "revive": "adunit-0000000000000002",
    },
};

// 开发环境使用测试广告位
if (isDevMode) {
    AdManager.getInstance().configure(TEST_AD_UNITS);
}
```

> **注意**：正式发布前必须替换为真实广告位 ID！

## 二、激励视频按位置显示

### 2.1 为什么按位置指定广告位？

1. **精准投放**：不同场景使用不同的广告位，便于在流量主后台分析各场景收益
2. **灵活控制**：可以单独调整某个位置的广告策略
3. **A/B 测试**：不同位置可以使用不同的广告位进行 A/B 测试
4. **清晰归因**：每个位置的收益来源清晰可查

### 2.2 使用方式

```typescript
// 看视频获得金币（使用 "gold" 位置对应的广告位）
AdManager.getInstance().showRewardedVideoAd(
    "gold",
    () => {
        // 用户看完广告，给予奖励
        console.log("获得 100 金币");
    },
    (err) => {
        // 广告加载失败
        console.error("广告播放失败", err);
    }
);

// 复活功能（使用 "revive" 位置对应的广告位）
AdManager.getInstance().showRewardedVideoAd(
    "revive",
    () => {
        // 用户看完广告，执行复活
        console.log("玩家复活");
    }
);
```

### 2.3 工作流程

```
玩家点击"看视频得金币"
        ↓
调用 showRewardedVideoAd("gold", onReward)
        ↓
查找 "gold" 对应的广告位 ID
        ↓
找到 adunit-yyyyyy
        ↓
显示该广告位的激励视频
        ↓
用户看完 → 执行 onReward 回调
用户关闭 → 不执行奖励
```

### 2.4 错误降级

如果某个位置的广告位加载失败，AdManager 会自动降级处理：

```typescript
// 如果 "gold" 未配置或广告加载失败
// AdManager 会直接执行 onReward 回调（降级处理）
// 确保用户体验不受影响
```

## 三、各广告类型配置

### 3.1 Banner 广告

```typescript
// 配置
banner: "adunit-xxxxxx"

// 显示（带自定义样式）
AdManager.getInstance().showBannerAd({
    left: 0,
    top: screenHeight - 100,  // 距离底部 100px
    width: screenWidth,       // 全屏宽度
});
```

### 3.2 激励视频广告（按位置）

```typescript
// 配置（按位置映射）
rewardedVideo: {
    "gold": "adunit-yyyyyy",
    "revive": "adunit-zzzzzz",
}

// 显示（指定位置）
AdManager.getInstance().showRewardedVideoAd(
    "gold",                     // 位置标识
    () => {                     // 看完回调
        console.log("获得 100 金币");
    },
    (err) => {                 // 失败回调（可选）
        console.error("广告失败", err);
    }
);
```

### 3.3 插屏广告

```typescript
// 配置
interstitial: "adunit-bbbbbb"

// 显示
AdManager.getInstance().showInterstitialAd();
```

### 3.4 原生模板广告

```typescript
// 原生模板广告需要单独的广告位 ID
const customAd = AdManager.getInstance().createCustomAd(
    "adunit-custom-xxxx",  // 原生模板广告位 ID
    {
        left: 0,
        top: 100,
        width: 300,
        fixed: true,  // 是否固定位置
    }
);
```

## 四、广告位 ID 获取流程

### 4.1 开通流量主

**条件**：
1. 累计独立访问用户（UV）不低于 1000
2. 小游戏类目符合要求
3. 小游戏已发布上线

**流程**：
1. 登录微信公众平台（mp.weixin.qq.com）
2. 进入「流量主」→「开通」
3. 填写资质信息（个人/企业）
4. 等待审核（1-3 个工作日）

### 4.2 创建广告位

1. 进入「流量主」→「广告管理」
2. 点击「新建广告位」
3. 选择广告类型（Banner / 激励视频 / 插屏 / 原生模板）
4. 填写广告位名称（便于管理）
5. 获得广告位 ID（adUnitId）

### 4.3 广告位命名建议

```
Banner 广告位：
- 名称：主界面底部Banner
- adUnitId: adunit-xxxxxx

激励视频广告位：
- 名称：看视频获得金币
- 位置标识: "gold"
- adUnitId: adunit-yyyyyy

- 名称：复活功能
- 位置标识: "revive"
- adUnitId: adunit-zzzzzz

插屏广告位：
- 名称：关卡结束插屏
- adUnitId: adunit-bbbbbb
```

## 五、注意事项

### 5.1 发布前检查清单

- [ ] 所有测试广告位 ID 已替换为真实 ID
- [ ] 激励视频已配置位置映射（rewardedVideo 对象）
- [ ] 每个位置都对应有效的广告位 ID
- [ ] Banner 广告位 ID 已配置（如果使用 Banner）
- [ ] 插屏广告位 ID 已配置（如果使用插屏）
- [ ] 已测试各位置的广告展示和回调逻辑

### 5.2 常见错误

**错误 1：位置未配置**
```
[AdManager] 位置 "gold" 未配置激励视频广告位
```
解决：检查 `configure()` 中 `rewardedVideo` 是否包含该位置。

**错误 2：广告位 ID 错误（errCode: 1003）**
```
[AdManager] 激励视频广告加载失败: {errCode: 1003}
```
解决：检查广告位 ID 是否正确，是否已开通流量主。

**错误 3：广告未填充（errCode: 1004）**
```
[AdManager] 激励视频广告加载失败: {errCode: 1004}
```
解决：这是正常情况，AdManager 会自动降级处理（直接给予奖励）。

### 5.3 调试技巧

```typescript
// 在开发阶段开启详细日志
if (isDevMode) {
    // 查看已配置的位置
    const positions = AdManager.getInstance().getRewardedVideoAdUnit("gold");
    console.log("gold 位置广告位:", positions);
}
```

## 六、示例：完整接入代码

```typescript
import { AdManager } from './docs/wechat-ads/AdManager';

export class GameManager extends Component {
    onLoad() {
        // 1. 配置广告位
        this.initAds();

        // 2. 显示 Banner 广告（延迟 1 秒，提升用户体验）
        this.scheduleOnce(() => {
            AdManager.getInstance().showBannerAd();
        }, 1.0);
    }

    private initAds() {
        const isDev = true; // 根据环境判断

        if (isDev) {
            // 开发环境：使用测试广告位
            AdManager.getInstance().configure({
                banner: "adunit-0000000000000000",
                interstitial: "adunit-0000000000000003",
                rewardedVideo: {
                    "gold": "adunit-0000000000000001",
                    "revive": "adunit-0000000000000002",
                },
            });
        } else {
            // 正式环境：使用真实广告位
            AdManager.getInstance().configure({
                banner: "adunit-your-banner-id",
                interstitial: "adunit-your-interstitial-id",
                rewardedVideo: {
                    "gold": "adunit-your-gold-ad",
                    "revive": "adunit-your-revive-ad",
                    "speedUp": "adunit-your-speedup-ad",
                },
            });
        }
    }

    /**
     * 看视频获得金币（位置：gold）
     */
    onWatchVideoForGold() {
        AdManager.getInstance().showRewardedVideoAd(
            "gold",
            () => {
                // 用户看完广告，给予奖励
                console.log("奖励 100 金币");
                this.addGold(100);
                this.showToast("获得 100 金币！");
            },
            (err) => {
                // 广告播放失败
                console.error("广告播放失败", err);
                this.showToast("广告加载失败，请稍后再试");
            }
        );
    }

    /**
     * 复活功能（位置：revive）
     */
    onRevive() {
        AdManager.getInstance().showRewardedVideoAd(
            "revive",
            () => {
                // 用户看完广告，执行复活
                console.log("玩家复活");
                this.revivePlayer();
            }
        );
    }

    /**
     * 加速功能（位置：speedUp）
     */
    onSpeedUp() {
        AdManager.getInstance().showRewardedVideoAd(
            "speedUp",
            () => {
                console.log("加速完成");
                this.applySpeedUp();
            }
        );
    }

    private addGold(amount: number) {
        // 添加金币逻辑
    }

    private showToast(message: string) {
        // 显示提示
    }

    private revivePlayer() {
        // 复活逻辑
    }

    private applySpeedUp() {
        // 加速逻辑
    }

    onDestroy() {
        // 销毁所有广告
        AdManager.getInstance().destroyAllAds();
    }
}
```
